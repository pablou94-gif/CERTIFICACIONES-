# CERTIFICACIONES-
Certificados y formacion para el analisis de datos

Nota sobre calificaciones
Algunos certificados la calificación se expresa en escala 1,0–7,0 (siendo 7,0 la nota máxima y 4,0 el mínimo habitual de aprobación). (Chile) 
Otros certificados los realicé en españa (sin esa escala 1–7).
